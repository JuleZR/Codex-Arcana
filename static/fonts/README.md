# Lokale Schriftarten

Dieses Verzeichnis enthaelt lokal gehostete Schriftarten fuer das Projekt.

## Enthaltene Fonts
- `cinzel/Cinzel-wght.ttf`
  - Quelle: Google Fonts (GitHub: `google/fonts`)
  - Lizenz: SIL Open Font License 1.1 (`cinzel/OFL.txt`)

- `cormorant_garamond/CormorantGaramond-wght.ttf`
  - Quelle: Google Fonts (GitHub: `google/fonts`)
  - Lizenz: SIL Open Font License 1.1 (`cormorant_garamond/OFL.txt`)

## Hinweis
- Bei Weitergabe/Deployment die jeweiligen Lizenzdateien mit ausliefern.
